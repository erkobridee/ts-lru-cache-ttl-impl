# ts-lru-cache-ttl-impl

Least Recently Used (LRU) cache with Time-to-Live (TTL) support [implemented in TypeScript](/src/lru-cache.ts)

## dev dependencies

```

  @types/jest \
  @types/node \
  esbuild \
  typescript \
  vitest
```
